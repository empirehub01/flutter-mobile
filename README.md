# Digital Daily Market 📊

A modern crypto trading platform with real-time candlestick charts, portfolio tracking, and paper trading capabilities.

## Features

✨ **Core Features**
- Real-time candlestick/price charts
- Buy/Sell order interface
- Portfolio tracking and analytics
- Market data feed (Binance, Kraken, Moonpay, Coinmama)
- Secure user authentication (JWT)
- Exchange API integration
- Paper trading mode (simulation)
- Modern glassmorphism UI with dark theme

## Tech Stack

### Frontend
- **Framework**: Next.js 14+ with React 18
- **Styling**: Tailwind CSS
- **Charts**: TradingView Lightweight Charts
- **State Management**: Redux Toolkit
- **HTTP Client**: Axios
- **Form Handling**: React Hook Form

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT + bcrypt
- **Real-time**: WebSocket (Socket.io)
- **API**: RESTful + GraphQL ready

### DevOps & Tools
- Docker & Docker Compose
- GitHub Actions CI/CD
- ESLint & Prettier
- Jest & Supertest
